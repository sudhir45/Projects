## General info
Model redicts what will be the House Prices using various factors like Parking, Garrage Lot Area and etc.

Algorithms used:
  1. Linear Regression
  2. Ridge Regression

Techniques like Imputing, One-Hot Encoding, Scaling were also used.
  
## Technologies
* Python
* Sklearn
* Jupyter Notebook
* Joblib
